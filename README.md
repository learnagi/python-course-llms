# Python LLM开发课程

本课程专注于使用Python开发大语言模型（LLM）应用，从基础环境搭建到高级应用开发，系统地介绍相关知识和实践。

## 课程目标

- 掌握Python编程基础
- 理解API和Web开发概念
- 学习LLM应用开发技术
- 掌握提示词工程技巧
- 构建完整的LLM应用
